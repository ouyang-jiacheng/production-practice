# Код для практики производственной стажировки

## Описание проекта
Этот репозиторий содержит программы, написанные для выполнения заданий производственной стажировки.

### Название программы: Калькулятор производительности смен (shift_calculation.py)
Эта программа была разработана для нефтехимической компании для автоматического расчета производительности каждой смены на основе ежедневных производственных данных и определения смены с наибольшей производительностью. Программа считывает информацию из пользовательского ввода или предопределенных данных, демонстрируя базовую логику обработки данных.

### Назначение
В компании это может помочь отделу планирования производства быстро рассчитать производительность смен, уменьшить ошибки ручных расчетов и повысить эффективность.
